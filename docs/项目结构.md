## 核心模块

### `src/db`

- `mod.rs`：数据库连接与迁移入口
- `btcc_wallet.rs`：BTCC 钱包数据读写、密码校验、加密解密、查询接口

### `src/wallet`

- `explorer.rs`：区块浏览器 API 客户端
- `keys.rs`：钱包生成、助记词恢复、WIF 恢复、靓号地址生成
- `pbe.rs`：本地加密方案
- `tx.rs`：单笔交易与批量交易构建、签名、地址校验
- `mod.rs`：统一导出对外接口

## 桌面端页面

### `wallet_list.rs`

负责：

- 钱包列表展示
- 查询
- 创建入口
- 导入入口
- 编辑 / 删除 / 导出
- 接收弹层
- 地址复制提示

### `vanity_generator.rs`

负责：

- BTCC 靓号地址生成页面
- 前缀 / 后缀 / 前后缀组合模式
- 线程数配置
- 命中结果展示
- 最近命中记录展示

### `wallet_generator.rs`

负责：

- 单次离线生成钱包
- 显示地址、派生路径、私钥、助记词
- 手动复制结果

### `wallet_manager.rs`

负责：

- 单笔发送
- 余额查询
- 签名预览
- 广播交易
- TXID 展示、复制、跳转浏览器

### `batch_send.rs`

负责：

- 批量转账
- 地址解析与校验
- 简单模式 / 专家模式
- 签名预览
- 广播交易

### `donate_window.rs`

负责：

- 打赏地址与二维码展示

## 导航关系

- `title_bar.rs`：顶栏菜单定义，包含 `BTCC`、`靓号生成`、`帮助`、`打赏`
- `dashboard.rs`：主页面路由与页面切换
- `btcc/mod.rs`：BTCC 相关页面模块导出
