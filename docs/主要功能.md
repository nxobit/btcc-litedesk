# 主要功能

本文档按当前代码状态整理 `BTCC Litedesk` 已实现的主要功能。

## 钱包列表

- 多钱包列表展示，按 `created_at DESC, id DESC` 倒序显示
- 钱包名称模糊查询
- 钱包地址完整匹配查询
- 地址复制提示
- 地址与余额隐藏 / 显示开关
- 总余额卡片显示 / 隐藏
- 行 hover 与选中态
- 从列表进入发送、接收、编辑、导出、删除

## 靓号生成

- 独立 `靓号生成` 页面，入口位于顶栏 `帮助` 前
- 页面风格与钱包列表页保持一致：页头卡片、统计卡片、参数区、结果区、记录区
- 支持三种匹配模式：
  - 前缀
  - 后缀
  - 前后缀组合
- 支持自定义并发线程数，范围 `1..=64`
- 命中后展示：
  - 网络
  - BTCC 地址
  - 派生路径
  - WIF 私钥
  - 助记词
  - 累计尝试次数
- 最近命中记录展示最近 8 条结果
- 页面只生成和展示结果，不自动写入钱包库

## 创建钱包

### 第一步

- 生成 12 个英文助记词
- 显示钱包名称、BTCC 地址、派生路径
- 地址支持复制
- 支持备注
- 输入钱包初始密码

### 第二步

- 随机抽取 3 个助记词做二次验证
- 支持 `关闭`
- 支持 `上一步`
- 支持 `验证并保存`

### 校验

- 单词为空时提示第几个单词未输入
- 单词错误时提示第几个单词不正确
- 密码为空时提示
- 密码长度不足时提示

## 导入钱包

- 支持 12 个英文助记词导入
- 助记词数量必须正确
- 每个助记词必须是合法英文单词
- 支持钱包名称、备注、钱包初始密码
- 已存在的钱包地址不会重复保存

## 单笔发送

- 选择发送钱包
- 查询确认余额、未确认余额、UTXO
- 收款地址校验
- 金额校验
- 点击 `发送` 先生成签名交易
- 弹出签名预览
- 预览确认后再广播
- 广播成功后显示 TXID
- 支持复制 TXID
- 支持跳转区块链浏览器

## 批量转账

### 基本能力

- 仅可选择有余额的钱包作为发送钱包
- 简单模式
- 专家模式
- 页内密码弹层
- 签名预览弹层

### 简单模式

- 每行一个钱包地址
- 统一金额输入

### 专家模式

- 每行格式：`钱包地址,金额`

### 批量校验

- 地址为空时报错
- 地址错误时提示第几个钱包地址有误
- 专家模式格式错误时提示第几行格式错误
- 专家模式金额错误时提示第几行金额错误
- 重复地址自动去重
- 单次最多 50 个地址
- 文本框下方显示当前非空行数

### 广播流程

- 点击 `发送`
- 先生成签名交易
- 弹出签名预览
- 点击 `广播交易`
- 广播成功后显示：
  - TXID
  - 复制
  - 区块链浏览器跳转

## 接收

- 页内弹层显示
- 显示钱包名称
- 显示收款地址
- 显示二维码 PNG
- 支持手机扫码
- 支持复制地址

## 打赏

- 顶栏独立 `打赏` 菜单
- 显示打赏地址与二维码

## 钱包安全

- 首次使用前要求设置钱包密码
- 未解锁前不显示 `BTCC` 菜单
- 助记词与 WIF 私钥写库前加密
- 密码校验使用 Argon2
- 导出前需要再次输入密码

## 查询与隐私

- 钱包列表支持数据库查询
- 地址隐藏时只显示前 5 位和后 5 位
- 余额隐藏时显示 `******`
- 总余额卡片支持显示 / 隐藏

## macOS 相关

- 初始窗口大小单独调整为 `1200 x 800`
- 全屏状态下提供 `退出全屏`
- Dock 重新点击可重新打开主窗口
- 打包脚本支持从 `assets/app.png` 生成 `app.icns`
- 当前 macOS 为未签名测试版

## 当前限制

- 当前只支持 `cc1q` 地址体系
- 靓号后缀仅支持 Bech32 字符集
- 靓号长度越长，耗时越高
- 批量转账单次最多 50 个地址
- 默认费率为 `5 sat/vB`
- macOS 对外分发仍建议签名与 notarization
## PBE 加密升级补充

- 本地钱包密文当前使用 `PBKDF2-HMAC-SHA512 + AES-256-CBC + 20000` 次迭代
- 兼容历史 `1000` 次迭代密文
- 旧密文会在钱包解锁成功或单条密钥解密成功后静默重写为新格式
- 详细说明见 [install/pbe-upgrade.md](./install/pbe-upgrade.md)
