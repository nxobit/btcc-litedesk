# CC-STAMP NFT 稀有度说明

## 说明目的

本页中的稀有度排序，仅用于帮助查看 21000 个 CC-STAMP NFT 在当前属性分布下的相对少见程度。

它不是价格评估模型，也不是价值判断依据。

## 稀有度算法

当前版本会对每个 NFT 的多个属性分别统计出现频次，然后把这些属性的少见程度汇总成一个综合分数。

参与统计的属性包括：

- `body_index`
- `head_index`
- `head_name`
- `eye`
- `eye_color`
- `rarity`
- `grad`
- `body_color`
- `head_hue`

对单个属性值，出现次数越少，说明这个属性越少见。

系统使用的综合分数公式为：

`rarity_score = Σ log(total_supply / trait_count)`

含义：

- `total_supply` 是总量，当前为 `21000`
- `trait_count` 是某个属性值在 21000 个 NFT 中出现的次数
- 某个属性值越少见，`trait_count` 越小，该项得分越高
- 一个 NFT 拥有的少见属性越多，综合分数通常越高

## 排序方式

页面里的排序含义如下：

- `编号从小到大`：按 NFT 编号正序
- `编号从大到小`：按 NFT 编号倒序
- `最常见优先`：按稀有度分数从低到高
- `最稀有优先`：按稀有度分数从高到低

## 需要注意

稀有度分数只反映属性组合在当前集合中的统计稀缺性，不代表：

- 市场价格
- 实际成交价值
- 用户偏好
- 艺术价值
- 收藏价值
- 流动性

因此，稀有度结果不能作为任何价值、投资、收益或交易决策的判断依据。

## 使用建议

更合理的使用方式是：

- 用它快速定位少见属性组合
- 用它辅助浏览和筛选 NFT
- 把它当作集合内部的统计参考，而不是价格结论
